Caponi Art Park

The Caponi Art Park application is an interactive, mobile friendly web app that uses geolocation and a Google Map overlay to locate a guest within the park as well as mark significant or nearby sculptures. Guests are able to select markers from the map to learn more about each sculpture through associated photos, writings, anecdotes, or poems. The application also allows guests to see upcoming events and sign up for the park's email list.

Slack Overflow

Slack Overflow, like Stack Overflow, allows users to search for and ask common development questions. The application is targeted toward current Prime students and addresses questions that are most specific to Prime assignments and technologies.
